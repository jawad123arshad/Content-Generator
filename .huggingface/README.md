# 🤖 MLOps Content Generator

A professional content generation app using FLAN-T5.

## Features
- Product descriptions
- Creative stories
- Technical explanations
- Professional emails

## Usage
1. Select model size (base recommended)
2. Load model (first time may take 2-3 minutes)
3. Enter your prompt
4. Click Generate
